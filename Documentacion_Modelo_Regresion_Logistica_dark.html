DATASET:User_Data

El conjunto de datos contiene informacin sobre usuarios potenciales de un producto o servicio. Contiene cuatro caractersticas de entrada: ID de usuario, Gnero, Edad y Salario estimado, que se utilizan para predecir si el usuario compr el producto, indicado por la columna de salida o destino "Comprado". El ID de usuario es un identificador nico asignado a cada usuario, mientras que el Gnero es el gnero del usuario, que puede ser masculino o femenino. La Edad es la edad del usuario en aos y el Salario estimado es una estimacin de su salario anual. El conjunto de datos puede utilizarse en tareas de clasificacin binaria para determinar si es probable que un usuario compre un producto o servicio en particular. Las caractersticas proporcionadas podran utilizarse para crear un modelo que prediga la probabilidad de que un usuario compre el producto en funcin de su edad, gnero y salario estimado.

La regresin logstica es una herramienta estadstica para predecir la probabilidad de que ocurra un evento, basndose en variables independientes. Se usa principalmente para problemas de clasificacin donde la variable de resultado es categrica, especialmente binaria (dos posibles resultados como s/no, 0/1). Su aplicacin incluye anlisis explicativos y predictivos en campos como la medicina, epidemiologa, finanzas y deteccin de fraudes.

Caractersticas principales

1. Prediccin: Estima la probabilidad de que una variable dependiente pertenezca a una categora especfica.
2. Resultados binarios: Es especialmente til cuando la variable de resultado solo puede tomar dos valores (es una variable dicotmica o binomial).
3. Funcin logstica (logit): Utiliza una funcin logstica para modelar la relacin, lo que resulta en una curva en forma de "S" y asegura que la prediccin de la probabilidad siempre est entre 0 y 1).
4. Anlisis de mltiples variables: Puede analizar el efecto de varias variables independientes (cuantitativas o cualitativas) sobre la variable    dependiente.
5. Funcin explicativa: Permite entender cmo las variables independientes

influyen en la probabilidad del resultado.

La curva ROC

La curva ROC en machine learning es una grfica que evala el rendimiento de un modelo de clasificacin binaria al mostrar la relacin entre la tasa de verdaderos positivos (TPR) y la tasa de falsos positivos (FPR) a travs de diferentes umbrales de decisin. Sirve para evaluar qu tan bien un modelo puede distinguir entre dos clases (por ejemplo, positivo vs. negativo), donde una curva cercana a la esquina superior izquierda indica un mejor rendimiento.

# Visualizacin de las regiones de decisin
def plot_decision_regions(X, y, classifier, resolution=0.02):
    # Configurar generador de marcadores y mapa de colores
    markers = ('s', 'x', 'o', '^', 'v')
    colors = ('red', 'blue', 'lightgreen', 'gray', 'cyan')

    # Mapa de colores
    from matplotlib.colors import ListedColormap
    cmap = ListedColormap(colors[:len(np.unique(y))])

    # Graficar la superficie de decisin
    x1_min, x1_max = X[:, 0].min() - 1, X[:, 0].max() + 1
    x2_min, x2_max = X[:, 1].min() - 1, X[:, 1].max() + 1
    xx1, xx2 = np.meshgrid(np.arange(x1_min, x1_max, resolution),
                           np.arange(x2_min, x2_max, resolution))
    Z = classifier.predict(np.array([xx1.ravel(), xx2.ravel()]).T)
    Z = Z.reshape(xx1.shape)

    plt.contourf(xx1, xx2, Z, alpha=0.4, cmap=cmap)
    plt.xlim(xx1.min(), xx1.max())
    plt.ylim(xx2.min(), xx2.max())

    # Graficar muestras de clase
    for idx, cl in enumerate(np.unique(y)):
        plt.scatter(x=X[y == cl, 0], y=X[y == cl, 1],
                   alpha=0.8, c=cmap(idx),
                   marker=markers[idx], label=cl)

    plt.xlabel('Edad (escalada)')
    plt.ylabel('Salario Estimado (escalado)')
    plt.legend(loc='upper left')
    plt.title('Regiones de Decisin - Regresin Logstica')

# Sistema con anlisis de confianza
class SistemaRecomendacion:
    def __init__(self, modelo_path, scaler_path):
        self.modelo = joblib.load(modelo_path)
        self.scaler = joblib.load(scaler_path)
        self.umbral_confianza = 0.6  # Umbral mnimo de confianza

    def analizar_cliente(self, edad, salario, nombre=""):
        # Preparar datos
        datos_cliente = np.array([[edad, salario]])
        datos_escalados = self.scaler.transform(datos_cliente)

        # Predecir
        prediccion = self.modelo.predict(datos_escalados)[0]
        probabilidades = self.modelo.predict_proba(datos_escalados)[0]
        confianza = max(probabilidades)

        # Determinar recomendacin
        if prediccion == 1 and confianza >= self.umbral_confianza:
            recomendacion = "CLIENTE PROMETEDOR - Invertir en marketing"
            accion = "Contactar con ofertas personalizadas"
        elif prediccion == 1 and confianza < self.umbral_confianza:
            recomendacion = "CLIENTE POTENCIAL - Monitorear"
            accion = "Enviar comunicaciones generales"
        else:
            recomendacion = "CLIENTE ESTNDAR - Mantener contacto bsico"
            accion = "Incluir en mailing list general"

        return {
            'nombre': nombre,
            'edad': edad,
            'salario': salario,
            'prediccion': prediccion,
            'probabilidad_compra': probabilidades[1],
            'confianza': confianza,
            'recomendacion': recomendacion,
            'accion': accion
        }

    def analizar_lote_clientes(self, lista_clientes):
        resultados = []
        for cliente in lista_clientes:
            resultado = self.analizar_cliente(**cliente)
            resultados.append(resultado)
        return resultados
